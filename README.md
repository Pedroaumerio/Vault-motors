## 🧠 Sobre o projeto

O **Vault Motors** é uma aplicação web desenvolvida para simular a experiência de uma plataforma exclusiva de veículos de luxo.

O projeto foca em:
- Interface sofisticada
- Experiência do usuário (UX)
- Design futurista (cyberpunk)
- Componentização e organização de código

---

## 🖥️ Funcionalidades

- 🏎️ Catálogo de veículos premium  
- ⭐ Sistema de favoritos  
- 👤 Página de perfil do usuário  
- 📊 Especificações técnicas detalhadas  
- 🎨 Interface moderna com efeitos visuais avançados  

---

## 🚀 Tecnologias utilizadas

- React.js  
- Next.js  
- Tailwind CSS  
- TypeScript  
- Vercel (deploy)
